
# Pilot Restaurant UI

Compliance dashboard for restaurant owners. Documents, calendar, inspection checklist, and AI assistant. The original project is available at https://www.figma.com/design/ipKXTuzdlq2dy1S5xQZrPo/Pilot-UI-for-Restaurant-Owners.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
